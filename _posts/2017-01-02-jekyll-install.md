## 博客使用 jekyll 

1. 安装 ruby ( 同时会安装gem )

下载地址 `https://rubyinstaller.org/downloads/` 

`ruby -v` 查看 ruby 版本

2. 安装 jekyll 

`gem install jekyll`

使用方式详见 `https://www.jekyll.com.cn/`

3. 如果出错，需要安装 bundle

`gem install bundle`
`bundle install`
`bundle exec jekyll serve`
